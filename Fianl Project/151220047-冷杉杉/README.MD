### 作业说明

* 图形界面
	葫芦娃和爷爷在左边排成长蛇阵，蝎子精和蛇精带着6个小喽在右边排成长蛇阵。他们分别用不同颜色的圆来表示。按下空格后，他们朝对方杀去。葫芦娃和爷爷在看到前面是坏蛋后开始在原地对对面进行社会主义教育。坏蛋在了解到自己的罪孽后原地消失。葫芦娃和爷爷继续前进直到坏蛋全部接受社会主义教育后停止。
	
* 文件结构

	生成的.jar在target目录下，所需的图片在resources目录下。源代码在main/java/default1下。由于不用maven时文件的编码为gbk，用了maven后文件编码变为utf-8，且无法变回来，所以有部分注释里的中文为乱码。
	另外，.txt是存档文件，程序默认的存档文件为test.txt。只保留一个存档。
	
### 所用类的结构
	原有的几个类和样例里几个类基本不变。不过原有的那些类新增继承了player类，方便进行线程操作。
	我新增的类中，filetest类用于生成读取存档的图形界面。
	square类用来保存程序运行实时情况
	save类用来生成和读取存档
	savetest类是在没有用maven时测试存读档是否成功的。使用了maven后我的测试用例也只包含这一个核心功能的测试
