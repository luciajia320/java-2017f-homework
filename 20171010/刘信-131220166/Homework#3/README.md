# 本作业中所使用的面向对象概念
- 将生物视为接口，其下的具体对象通过该接口来体现它们作为生物的特性
- 将宇宙中的每个空间都视为对象，由于它们可以并且仅可以被一个生物占据，所以每个空间都拥有关于在这个空间上的生物的属性
# 仍存在的问题
- 不确定单个空间是否应该储存自己的位置信息，这个应该是在宇宙范畴上的
# 运行结果
## 输出
运行结束后会输出两个html文件，打开即可浏览。如需改变阵法，要main中改变Deployment即可。
