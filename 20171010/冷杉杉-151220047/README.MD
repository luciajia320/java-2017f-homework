#面向葫芦娃编程
##类的设计
###creature
总的父类。葫芦娃，蛇精等为其子类。
为了方便显示，修改了各子类的report方法
各显示一个代表实例的汉字
####shejing,xiezijing,yeye
具有基本结构的子类，通常只有一个实例
####xiaolouluo
有若干的实例
####Huluwa
有若干个实例，并有别的子类没有的属性--颜色
###position
与老师给的一样，没有变化
###queue
改成二维的
大小暂定为NxN，N=11
初始化参数为葫芦娃
其他的实例后面通过standinqueue添加进去
通过print方法输出
